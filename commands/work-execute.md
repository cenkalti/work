# Execute

You are working on a task defined in your CLAUDE.md. Follow these steps:

## Step 1: Read Your Task

Read the CLAUDE.md file in this directory to understand your goal and task assignment.

## Step 2: Start the Work Log

Create your work log at `workspace/log.md`. Use this structure:

```markdown
# Work Log

## Status: in_progress

## Task
<copy your task summary here>

## Progress

## What's Left

## Notes
```

## Step 3: Do the Work

Implement the task according to the acceptance criteria. As you work:
- Update the **Progress** section with what you did and key decisions
- Update **What's Left** with remaining items
- Add anything important to **Notes**
- Use `workspace/` for any scratch files or intermediate outputs you need

## Step 4: Finish Up

When done:
- Set status to `completed`
- Write a final summary in Progress
- Clear "What's Left" or note any follow-ups
- Make sure all acceptance criteria are met
